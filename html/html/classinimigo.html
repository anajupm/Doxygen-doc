Detailed Description

Classe que representa um inimigo no jogo de batalha naval.

A classe inimigo herda da classe jogador e adiciona métodos e variáveis específicas para gerenciar o comportamento do inimigo.

Método para registrar um disparo no tabuleiro do inimigo.

Registra um disparo no tabuleiro do inimigo na posição (x, y).

Parameters
xCoordenada X do disparo.
yCoordenada Y do disparo.

Se um navio for atingido, a posição é marcada como 'O' e a vida do inimigo (vidaIN) é incrementada. Se o disparo não atingir um navio, a posição é marcada como 'X'.

Método para imprimir todos os navios do inimigo.
